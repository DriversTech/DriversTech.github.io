Driver’s Tips


  • Unless you're a business, Microsoft will not ever want you to call them. It is a fake scam.
  • If a message feels urgent, threatening, or pushy, don’t click it.
  • Restart Wifi by unplugging your modem/router for 30 seconds, then plugging it back in.
  • Save your work regularly (Ctrl + S) to avoid losing changes.
  • If a program freezes, try Ctrl + Alt + Delete and use Task Manager to close it.
  • Reboot the computer once before assuming something is “broken.”
  • Back up important files at least once a month to an external drive or cloud service.
